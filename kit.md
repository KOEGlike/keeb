# Kit contents!

Here are all the parts that come with the kit! It should be enough for any keyboard you may want to make.

You're allowed to use other parts, but you'll have to source them yourself on your on money

Feel like something should be included? Let me know and it might just make it into the next edition, if that ever happens

- 1x [Orpheus Pico](https://orpheuspico.hackclub.com/)
- 104x MX-Style switches
- 2x [EC11](https://www.alibaba.com/product-detail/EC11-rotary-15-20mm-Meihua-Bing_1600985459709.html) Rotary encoders
- 104x white blank DSA keycaps 
- 104x [SK6812 MINI-E LEDs](https://www.lcsc.com/product-detail/C5149201.html?s_z=n_SK6812%2520MINI-E) ($5.55 if we order full reel)
- stabilizers for a full sized keyboard
- 1x 0.91 inch OLED display (the pin order is GND-VCC-SCL-SDA, MAKE SURE YOUR PCB MATCHES)
- 8x M3x16mm screws
- 8x [M3x5mx4mm heatset inserts](https://www.aliexpress.us/item/2255800046543591.html)
- 110x through-hole 1N4148 Diodes
