The kit comes with more than just a few keyswitches! It comes with RGB LEDs, rotary encoders, sliders, a buzzer, joysticks, and an OLED screen!

As with anything, make sure to ask in **#keeb** if you ever have any questions!

A lot of popular keyboards are open source, so you can look at them for inspiration.

## Finding Symbols and Footprints

A lot of these can already be found in the default KiCAD libraries or marbastlib, so you should look in these places first.

If you can't find what you are searching for in the above mentioned sources, you should search on [SnapEDA](https://www.snapeda.com/) or [Component Search Engine](https://componentsearchengine.com/).

## Finding 3D Models

You should search on [GrabCAD](https://grabcad.com/library) and sort by most popular, and you will most likely find the model that you are looking for.
